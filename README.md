##Features
- Download videos as MP4 or MP3
- Automatically chooses the highest quality availible. 
- 5 video download queue!
- Lightweight and easy to use!
- Fully open source; All source code to be found at https://github.com/aanderson1212/YTDL

## Requirements
- Python 3.10+
- yt-dlp (Included)
- ffmpeg.exe (NOT Included) ((If I find time, I can throw together an auto installer for ease of use. 
Until then, install it yourself or find me and ask for the file via USB :^)))
