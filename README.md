##YT-DOWNLOADER --- MIT Open Source License
-------------------------------------------------------------------------------------------------------------------------
##Features
- Download videos as MP4 or MP3
- Automatically chooses the highest quality availible. 
- 5 video download queue!
- Lightweight and easy to use!
- Fully open source; All source code to be found at https://github.com/aanderson1212/YTDL

## Requirements
- Python 3.10+
- yt-dlp (Included)
- ffmpeg.exe (**NOT** Included)

Install Python packages:
--------------------------------------------------------------------------------------------------------------------------
#bash
pip install -r requirements.txt
